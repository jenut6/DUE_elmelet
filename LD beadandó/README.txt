Feladat leiras
A feladat egy dolgozat kiértékelő mini program, ami a dolgozat pontszámát és az elért pontszámot kéri be.
A program kiszámolja a kapott pontszámból az elért átlagot és egy pontozási rendszeren keresztül kiértékeli azt. 
A program Python kódban készült ezáltal CMD-ből lehet futtatni!
Pontozási rendszer
5:  80% - 100%
4:  70% - 99%
3:  60% - 69%
2:  50% - 59%
1:  0% - 50%
